# 🛍️ Customer Analysis Project

## Overview
End-to-end customer analysis on 5,630 e-commerce customers using Python and Tableau.

## 📊 Key Findings
- 16.8% churn rate (948 out of 5,630 customers)
- 43% of customers are "At Risk" — urgent re-engagement needed
- Churn is highest in first 0-2 months of tenure
- Built churn prediction model with **89% accuracy**

## 🛠️ Tools Used
- Python (pandas, numpy, matplotlib, seaborn, scikit-learn)
- Tableau Public
- Jupyter Notebook
